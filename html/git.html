Git

  • git --version - выводит версию git.
  • git init - инциализация репозитория.
  • git config — настройки параметров конфигурации Git.
  • git status - состояние рабочего каталога и индекса.
  • git add <имя_файла> - добавление файлов/изменений в индекс (подготовка к коммиту).
  • git push - отправка коммитов на удалённый репозиторий (например, на GitHub).
  • git log - просмотр истории коммитов.
  • git pull - извлечение и слияние изменений.
  • git branch - управление ветками.
